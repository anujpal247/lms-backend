# LMS Backend

### Project setup

1. Clone the repository

```
  git clone https://github.com/anujpal247/base-node-express-template.git
```

2. Change directory

```
  cd lms-backend
```

3. Create `.env` file in root directory and add

```
  PORT=5000
  FRONTEND_URL=""
```

3. Install dependencies

```
  npm i 
```
